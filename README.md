# LoveMyLife Weight Companion, prototype

Clickable prototype of the LoveMyLife Weight Management (GLP-1) companion, patient app and doctor view. Mock data only, no backend.

Live: https://sethrankin.github.io/weight-companion-prototype/

Second vertical on the companion pattern, mirroring the Sinclair companion (sinclair-companion-prototype). Copy and config would come from Storyblok; per-patient logs from the new database. Numbers here are invented.
